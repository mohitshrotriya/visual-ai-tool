Automatically log all detected bugs into a Google Sheet. Every time a bug is found, a new row is added with test name, severity, description, diff percentage, and timestamp.

How to Setup

1. Go to sheets.google.com and create a new spreadsheet
2. Name it QAMS Bug Reports
3. Copy the Sheet ID from the URL:
docs.google.com/spreadsheets/d/SHEET_ID_HERE/edit
4. Paste the Sheet ID above and click Save
5. Make sure your server has credentials.json configured
